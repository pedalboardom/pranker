ABOUT IVANIZER

What is this?
Musician Ivan Anderson (known on YouTube as cyberattack) makes tier-ranking videos where he sorts various categories of guitar pedals into tiers — Ultra, High, Mid, and Low. The Ivanizer is an homage to that format, letting you build your own pedal tier lists using the same layout.
How to use
🔍Search — Type a brand or pedal name to find it, then tap a tier button (U/H/M/L) to add it.
✏️Title — Set a title for your tier list (e.g. "Every Reverb I've Ever Owned"). Shows up on the canvas and in the exported image.
📸Export — Renders your tier list as a PNG image with the graph paper background, ready to share.
💾Save — Downloads your tier list as a JSON file so you can back it up or share the raw data.
📂Load — Load a previously saved JSON tier list file.
🗑Clear — Remove all pedals from every tier and start fresh.
↔️Reorder — Drag pedals to reorder within a tier or move them between tiers. On mobile, long-press or swipe to start dragging.
